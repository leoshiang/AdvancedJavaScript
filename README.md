JavaScript 的物件是所有**魔法**的基礎。沒錯，對大部分的 JavaScript 開發者來說，程式碼就像跟符咒一樣難懂，但是當你掌握了 JavaScript 的物件特性之後，所有的符咒就跟白話文一樣淺顯易懂。
